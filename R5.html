Graphics and Design are important when creating a product. Especially when it comes to games. When I created my first game, though I had to use stock sprites with the given game engine, I ended up creating a custom icon in order to draw in people to play my game. This is reflected that creating eye-catching design can help people use your UI or website.


The use of typefaces, font sizes, spacing, and color help establish what the mood of the website show and how the user experience navigating through it. By having pleasing and colorful pallet, it helps the UX and give a good impression. Typefaces help give professionalism and cleanliness throughout the text. Proper spacing gives room for the user read through the text and help where the eye moves which space.
